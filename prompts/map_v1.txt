- If none, return "action_items": [].

Respond with a single valid JSON object matching this schema exactly:
{
  "chunk_summary": "3-5 sentence prose summary of this chunk",
  "action_items": [
    {
      "owner": "Name or 'unassigned'",
      "task": "Concise description",
